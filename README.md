## Installation

First, install [Yeoman](http://yeoman.io) and generator-vsv using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-vsv
```

Then generate your new project:

```bash
yo vsv
```
